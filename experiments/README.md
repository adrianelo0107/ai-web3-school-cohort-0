# Experiments

Use this folder for small, public-safe AI x Web3 experiments.

Good experiment examples:

- Prompt workflow for analyzing an on-chain project.
- Script that summarizes public protocol documentation.
- Mock agent flow for wallet risk explanation without real private keys.
- Product research workflow using public data only.

Each experiment should include:

- Goal
- Hypothesis
- Steps
- Result
- What changed in my product thinking
